
### Getting Started (In C# directory)
---------------
**Env**
  - OS: Arch Linux 
  -  Editor: Doom Emacs
  ---------------
  **Setup**

   

    dotnet new console --name <your name project>

   or use alias dn config via zshrc 
   
    dn new console --name <your name project>
    
  ---------------
  **Run**
  go to project directory and run this command

    dotnet run 
or alias dn

    dn run
